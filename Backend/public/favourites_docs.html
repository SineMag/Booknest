Favourites API

Public Route: https://booknestapi.netlify.app/favourites.html

üìò Introduction

The /favourites API allows users to save and manage their favourite accommodations on the BookNest platform. It supports creating favourites, listing all favourites per user, and removing a favourite.

üß© Data Model

Each favourite record has the following structure:

{
  id: number,
  userId: number,
  accommodationId: number
}

POST /favourites

Adds a new favourite record for a user.

DELETE /favourites/:id

Deletes a favourite by its id and returns the deleted record.

‚úî Example

‚úî Example Response

  "id": 21,
  "userId": 5,
  "accommodationId": 10

üõ† Related Database Schema

CREATE TABLE IF NOT EXISTS favourites (
  id SERIAL PRIMARY KEY,
  userId INTEGER NOT NULL,
  accommodationId INTEGER NOT NULL,
  FOREIGN KEY (userId) REFERENCES users(id) ON DELETE CASCADE,
  FOREIGN KEY (accommodationId) REFERENCES accomodations(id) ON DELETE CASCADE
);
