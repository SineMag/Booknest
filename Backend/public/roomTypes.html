RoomTypes API Documentation

üìò Introduction

The /room_types API allows you to create, read, update, and delete room type records in the BookNest platform. This page shows you how each route works, what data you should send, and what responses to expect.

üß© Data Model

Each room type has the following structure:

{
  id: number,
  name: string,
  description: string,
  pricePerPersonPerNight: number,
  beds: number
}
