üìò Introduction

The /admins API manages administrative accounts for the BookNest platform. This includes creating admins, updating profiles, viewing all admins, and deleting admin accounts.

üß© Data Model

An admin has the following structure:

{
  id: number,
  firstName: string,
  lastName: string,
  emailAddress: string,
  physicalAddress: string,
  phoneNumber: string,
  profilePicUrl?: string,
  password: string,
  createdAt: Date
}
