Route: /accomodations

üìò Introduction

The /accomodations API allows you to create, read, update, and delete accommodation listings in the BookNest platform. This page shows you how each route works, what data you should send, and what responses to expect.

üß© Data Model

Each accommodation has the following structure:

{
  id: number,
  name: string,
  description: string,
  imageGallery: string[],
  amenities: string[],
  physicalAddress: string,
  phoneNumber: string,
  emailAddress: string,
  roomTypes: string[],
  rating: number,
  pricePerNight: number
}
