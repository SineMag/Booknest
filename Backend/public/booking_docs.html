Bookings API Documentation

Route: /bookings

Public Page: https://booknestapi.netlify.app/booking.html

üìò Introduction

The /bookings API allows creating, fetching, updating, and deleting booking records. Each booking connects a user to an accommodation and a room type.

üß© Data Model

A booking contains the following fields:

{
  id: number,
  checkInDate: string (YYYY-MM-DD),
  checkOutDate: string (YYYY-MM-DD),
  numberOfGuests: number,
  totalPrice: number,
  specialRequest: string | null,
  status: string,            // e.g. "pending", "confirmed", "cancelled"
  userId: number,
  accommodationId: number,
  roomTypeId: number
}
