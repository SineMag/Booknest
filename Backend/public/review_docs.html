Reviews API Documentation

Route: /reviews

üìò Introduction

The /reviews API allows users to create, read, update, and delete reviews related to accommodations on the BookNest platform. This documentation explains each route, what it expects, and what it returns.

üß© Data Model

Each review has the following structure:

{
  id: number,
  starRatings: number,
  reviewer: string,
  reviewText: string,
  createdAt: Date,
  userId: number,
  accomodationId: number
}

GET /reviews/:id

Returns a specific review by its ID.

‚úî Example

‚úî Example
